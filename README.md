## How to seed initial data ?

Create a `seeder.json` file before running the project

```json
{
    "quantity": "11009",
    "code": "BTC",
    "from_code": "",
    "from_quantity": "",
    "reference": {
        "value": "5000.0",
        "currency": "CAD"
    },
    "date": "YYYY-MM-DD"
}
```

Software will delete it once done
